# Restaurant Ordering App

A simple, interactive front-web application that allows users to order food from a diner menu.

## Features

* **Interactive Menu:** Displays a list of available items, including Pizza, Hamburgers, and more!


* **Shopping Cart:** Users can add items to their order, and the app will automatically calculate the total price.


* **Modify Order:** Includes the ability to remove specific items from the cart before completing the purchase.


* **Checkout Modal:** Features a clean payment form that pops up when the user is ready to complete their order.


* **Order Confirmation:** Shows a personalized confirmation message once the payment form is successfully submitted.



## Tech Stack

* **HTML5:** Provides the semantic structure for the app, including the menu container, order summary, and payment modal.


* **CSS3:** Handles the styling, utilizing flexbox for layout, custom fonts, and modal overlays.


* **Vanilla JavaScript:** Manages the dynamic rendering of menu items, event listeners for user interactions, and cart calculations.



## Project Structure

* `index.html`: The main structural file that links the stylesheet and scripts.


* `styles.css`: Contains all visual design elements and layout rules for the application.


* `index.js`: The core script that handles logic, such as adding/removing items, updating the DOM, and toggling the payment modal.


* `data.js`: A module containing the array of menu item data (names, ingredients, prices, and emojis).



## How to Run Locally

1. Clone this repository to your local machine.
2. Ensure all files are located in the same directory.
3. Open `index.html` in your preferred web browser to view and interact with the app.
